## 🚀 วิธีติดตั้งและรัน

### ข้อกำหนด
- **Python 3.8+** (ดาวน์โหลดที่ https://python.org)

### ขั้นตอน

**1. ติดตั้ง Python packages**
```bash
pip install -r requirements.txt
```

**2. รันแอพพลิเคชัน**
```bash
python app.py
```

**3. เปิดเบราว์เซอร์**
```
http://localhost:5000
```

> 💡 ข้อมูลตัวอย่างจะถูกเพิ่มอัตโนมัติเมื่อรันครั้งแรก

---

## 📱 วิธีใช้งานบนมือถือ

1. รันแอพบนคอมพิวเตอร์ในเครือข่าย WiFi เดียวกัน
2. หา IP ของคอมพิวเตอร์ (เช่น `ipconfig` บน Windows, `ifconfig` บน Mac/Linux)
3. เปิดเบราว์เซอร์มือถือและเข้า `http://[IP คอมพิวเตอร์]:5000`
   - ตัวอย่าง: `http://192.168.1.100:5000`

## ✨ ฟีเจอร์ทั้งหมด

### 🏠 หน้าหลัก (Dashboard)
- สถิติภาพรวม: จำนวนโครงการ รายได้รวม แรงงานวันนี้ งานล่าช้า
- แจ้งเตือนวัสดุใกล้หมด
- โครงการที่กำลังดำเนินการ
- รายการเงินล่าสุด

### 🏢 โครงการ
- ดูโครงการทั้งหมดพร้อม Progress Bar
- กรองตามสถานะ (กำลังก่อสร้าง / ล่าช้า / ตามแผน / เสร็จแล้ว)
- เพิ่ม / แก้ไข / ลบโครงการ
- ดูมูลค่าสัญญา วันส่งมอบ

### 👷 แรงงาน & ทีมงาน
- รายชื่อพนักงานพร้อมตำแหน่งและค่าแรง
- สถานะ: เข้างาน / ลาป่วย / ลากิจ / OT
- กรองตามสถานะ
- เพิ่มพนักงานใหม่พร้อมมอบหมายโครงการ

### 💰 การเงิน
- สรุปรายรับ-รายจ่าย และกำไรสุทธิ
- บันทึกรายการรับเงิน / จ่ายเงิน
- หมวดหมู่: ค่างวด / วัสดุ / ค่าแรง / เครื่องจักร / อื่น ๆ

### 🧱 วัสดุ & คลังสินค้า
- ติดตามปริมาณวัสดุในคลัง
- แจ้งเตือนเมื่อวัสดุต่ำกว่าขั้นต่ำ
- Progress Bar แสดงระดับสต็อก
- เพิ่ม / แก้ไข วัสดุ

### 🚜 เครื่องจักร & อุปกรณ์
- ติดตามสถานะเครื่องจักร: ว่าง / ใช้งาน / ซ่อมบำรุง / เสีย
- มอบหมายให้โครงการ

### ⛑️ ความปลอดภัย
- รายงานอุบัติเหตุ
- ระดับความรุนแรง: Near Miss / เล็กน้อย / ปานกลาง / รุนแรง

## 🗄️ ฐานข้อมูล

ใช้ **SQLite** — ไม่ต้องติดตั้งเพิ่ม ไฟล์ `data/buildmaster.db` จะถูกสร้างอัตโนมัติ

### ตาราง
| ตาราง | คำอธิบาย |
|-------|----------|
| projects | โครงการ |
| workers | พนักงาน |
| materials | วัสดุ |
| equipment | เครื่องจักร |
| transactions | รายการเงิน |
| safety_incidents | อุบัติเหตุ |
| attendance | การเข้างาน |

---

## 🔧 API Endpoints

| Method | Endpoint | คำอธิบาย |
|--------|----------|---------|
| GET | `/api/dashboard` | ข้อมูลหน้าหลัก |
| GET/POST | `/api/projects` | โครงการ |
| GET/PUT/DELETE | `/api/projects/:id` | โครงการรายการ |
| GET/POST | `/api/workers` | พนักงาน |
| PUT/DELETE | `/api/workers/:id` | พนักงานรายการ |
| GET/POST | `/api/materials` | วัสดุ |
| PUT/DELETE | `/api/materials/:id` | วัสดุรายการ |
| GET/POST | `/api/equipment` | เครื่องจักร |
| PUT/DELETE | `/api/equipment/:id` | เครื่องจักรรายการ |
| GET/POST | `/api/transactions` | รายการเงิน |
| DELETE | `/api/transactions/:id` | ลบรายการเงิน |
| GET/POST | `/api/safety` | อุบัติเหตุ |
